## Nome do Projeto

Fila_encadeada

## Autor

Patryck Henryck Moreira Silva

### Descrição

Trata-se de uma estrutura de dados simples, no caso uma fila Fila_encadeada
Este projeto foi realizado com o próposito avaliato da disciplina Engenharia de software

### Recursos

Tem como utilidade organizar números inteiros em uma fila encadeada

### Pré-requisitos

Conter dos 3 arquivos do projeto sendo eles: "main.c", "Fila_Encad.h", "Fila_Encad.c"
Compilar "main.c" e "Fila_Encad.c" juntos.
Utilizando o compilador para c de sua escolha

### Exemplos

Pode ser usado em qualquer tipo de cenário que se utilize uma fila
Sendo possível alterar o projeto para receber mais dados, tornando assim seu uso mais amplo

### Contribuição

Pode ser feita contribuições em relação a interface e manipulação de mais tipos de dados
